WaveAnalyzer

Interactive audio waveform analyzer using D3.js

View on GitHub

Project Overview

WaveAnalyzer is a tool for visualizing audio waveforms in real-time or from files. Perfect for audio analysis, AI audio projects, and sound design experimentation.

Code Example

    
    import waveAnalyzer from 'wave-analyzer';

    const audioFile = 'sample.wav';
    waveAnalyzer.load(audioFile).then(data => {
      waveAnalyzer.visualize(data, '#wave-chart');
    });
